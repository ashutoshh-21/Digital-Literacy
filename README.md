## 📌 About This Project

This repository is my Digital Literacy Portfolio, created as part of the **CSE0001 – Digital Literacy** course at **VIT Bhopal University**. The project is structured around the role of a *Student Digital Ambassador* and covers five core modules: digital awareness, professional online presence, coding & collaboration platforms, email communication, and cybercrime prevention.

> *"Digital literacy is not just about using technology — it's about using it wisely, safely, and professionally."*
